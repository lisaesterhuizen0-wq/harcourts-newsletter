# Harcourts Platinum — The Platinum Report

A monthly HTML email newsletter for **Harcourts Platinum**, a real estate agency serving the Helderberg basin in South Africa.

**Live preview:** https://lisaesterhuizen0-wq.github.io/harcourts-newsletter/

## Why I built this

The agency wanted a polished, on-brand monthly newsletter without paying for a full email-marketing platform. I built the template as a hand-coded HTML email and wired the dispatch into a Make.com workflow so a non-technical user can update the content and send the issue each month.

## What's in this repo

- `index.html` — the email template itself (table-based layout, inline styles, no JavaScript). Tested in the major email clients.

The Make.com automation that drives the actual send lives outside this repo.

## Tech

- HTML (table-based email layout)
- Inline CSS (required by most email clients)
- Make.com — for the monthly dispatch workflow

## How to preview locally

Open `index.html` in any browser, or visit the live preview link above.
